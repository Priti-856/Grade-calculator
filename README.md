**CGPA Calculator** using cpp
This is a helpful C++ tool for students to calculate their Cumulative Grade Point Average (CGPA). It uses the number of credits and grade points for each subject to figure out the final score.

**Features**
->Student Profile: Saves the student's name for a personalized result page.
->Flexible Subjects: Lets you enter as many subjects as you want.
->Weighted Math: Uses class credits to calculate the true weighted CGPA score.
->Clean Summary: Shows a clear report with the final numbers at the end.

**How to Run It**
->Make sure you have a C++ compiler (like g++) installed on your computer.
->Download the code file (for example, cgpa_calc.cpp).
->Open your terminal or command prompt.
->Compile the code with this command:
g++ cgpa_calc.cpp -o cgpa_calc

->Run the tool with this command:
./cgpa_calc

**How to Use It**
->Start the tool and type the student's name.
->Enter the total number of subjects you want to score.
->For each subject, type its name, credit hours, and your grade point.
->View your final CGPA printed under the results section.
